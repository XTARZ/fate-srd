# 入门导引

这是*命运浓缩版*，是我们对*命运核心系统*所整理的最简约形式。它是完整的角色扮演游戏；虽然其他书籍能够让你强化使用本系统，但并非必须的。

那么，来看看你需要*做*什么吧！

## 我需要什么来玩？

要玩命运浓缩版，你需要二到六名朋友（包括一名游戏主持人（GM））、一些骰子、一些指示物、书写工具、纸、可以写下短语的东西（如小纸条）。

当角色行动，命运浓缩版使用**命运骰**™。一颗命运骰是有两个<FateFont>0</FateFont>面、两个<FateFont>+</FateFont>面、两个<FateFont>-</FateFont>面的六面骰。至少要有一组四个骰子，但最好每个玩家都有一组。还有其他选项，像是使用标准的六面骰（1-2 =<FateFont>-</FateFont>、3-4 =<FateFont>0</FateFont>、5-6 =<FateFont>+</FateFont>）、命运牌组（改用卡片而非骰子）。但总之我们都会说是「掷骰」来简化说明。

> 如果你临时无法取得命运骰，你也可以使用本站的[线上掷骰器](../tools/fate-dice)。

### 给老手：从命运核心版改了什么

将一个系统的要点从近 300 页压缩到约 50 页的文字，确实会导致改动。此外，在本文写作时，*命运核心版*也已经八年了，所以预期也会有设计上的修正。特别是以下几点：

- [**压力格**改成了单点的](getting-start#%E5%A3%93%E5%8A%9B%E8%88%87%E5%BE%8C%E6%9E%9C)，减少混乱。
- 预设改为[「巴尔塞拉式（Balsera）」先攻](challenges-conflicts-and-contests#%E5%9B%9E%E5%90%88%E6%AC%A1%E5%BA%8F)（也可以称之「选取行动排序」或「爆米花式先攻」），而不用技能来决定回合顺序。
- [成长的功能有些改动](advancement)；我们取消了重要里程碑，取而代之地，我们对主要里程碑（作为突破点）进行微调。
- 我们取消了主动抵抗的概念，将它与**防御行动**分开。这产生了新的连锁反应，特别是[**克服**的平手结果](taking-action-rolling-dice#%E5%B9%B3%E6%89%8B)。
- [我们对**创造优势**进行了调整](taking-action-rolling-dice#%E5%89%B5%E9%80%A0%E5%84%AA%E5%8B%A2)，以便在发现现有未知**形象**时更清晰并且有更好的作用。
- **完全防御**改成可选，并对**防御行动**加以扩充以进行补偿。并加上[许多标准可选规则](optional-rules)。
